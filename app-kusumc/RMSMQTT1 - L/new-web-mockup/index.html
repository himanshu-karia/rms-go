Connection Help

Working Public Brokers:
  • broker.hivemq.com:8884 - HiveMQ (Secure WebSocket)
  • test.mosquitto.org:8081/mqtt - Mosquitto (Secure WebSocket)
  • broker.emqx.io:8083 - EMQX (WebSocket)
  • localhost:8080 - Local development
Connection Tips:
  • Use WSS (Secure WebSocket) for public brokers
  • Use WS (WebSocket) for local/non-secure brokers
  • Check browser console (F12) for detailed error messages
  • Some browsers block mixed content (HTTP on HTTPS sites)
  • Firewalls may block WebSocket connections
